# 52. 채권 심화 용어집

증권사 백엔드 개발자가 채권·채권평가 도메인을 온보딩할 때 반드시 숙지해야 할 핵심 용어를 종류·발행구조·가격수익률·위험민감도·거래 다섯 영역으로 분류하여 망라했다. 각 용어의 계산 관행과 시스템 구현 시 주의사항을 개발 포인트로 병기한다.

---

## 채권 종류

| 용어 | 영문/약어 | 뜻 | 개발 포인트 |
|------|-----------|-----|-------------|
| 국채 (국고채) | Government Bond / KTB | 기획재정부가 발행하는 대한민국 정부 채권. 3·5·10·20·30·50년물이 있음. | 국고채 단가는 금융투자협회(KOFIA) 수익률 고시 기준 적용. 종목코드 KR로 시작 |
| 통안채 | Monetary Stabilization Bond / MSB | 한국은행이 통화량 조절을 목적으로 발행하는 채권. 만기 14일~2년. | 만기 14일~91일은 할인채, 182일 이상은 이표채 구조 혼재. 만기유형 분기 처리 필요 |
| 지방채 | Local Government Bond | 지방자치단체가 재원 조달을 위해 발행하는 채권. | 발행기관 코드 관리 필요. 신용등급은 지자체 자체 등급이 아닌 보증기관 등급을 참조하는 경우 多 |
| 특수채 | Special Bond | 특별법에 근거한 공사·기관(예금보험공사, 한국전력 등)이 발행하는 채권. | 발행기관별 법적 근거가 다름. 발행기관 마스터 테이블에 법적 근거 필드 유지 권장 |
| 금융채 | Financial Bond | 은행·보험·카드사 등 금융기관이 발행하는 채권. 은행채·카드채 포함. | 발행사 업권(bank/card/insurance) 구분 필드 필요. 업권별 규제자본 산정 로직 상이 |
| 회사채 | Corporate Bond | 일반 기업이 자금 조달을 위해 발행하는 채권. | 신용등급 + 보증 여부 + 담보 여부 복합 관리. 무보증/보증/담보 구분 코드 설계 중요 |
| 전환사채 | Convertible Bond / CB | 일정 조건 하에 발행 회사 주식으로 전환 가능한 채권. | 전환가, 전환청구기간, 리픽싱 조건 별도 테이블 관리. 전환권 행사 시 채권→주식 계정 이관 처리 |
| 신주인수권부사채 | Bond with Warrant / BW | 신주인수권(워런트)이 부여된 채권. 워런트 분리 후 각각 거래 가능. | 채권 원본과 워런트를 별개 종목으로 관리. 분리형 BW는 워런트 별도 ISIN 발급됨 |
| 교환사채 | Exchangeable Bond / EB | 발행사가 보유한 타사 주식으로 교환 가능한 채권. CB와 달리 신주 발행 없음. | 교환 대상 주식 종목 코드·수량·교환가 관리. 교환 이벤트 발생 시 보유 주식 처분 처리 |
| 신종자본증권 (영구채) | Hybrid Capital Security / Perpetual Bond | 만기가 없거나 매우 길고, 이자 지급 유예·원금 상각 조건이 붙은 채권. 자본으로 회계 처리 가능. | 회계상 자본/부채 분류 구분 필드 필수. 이자 지급 유예(deferral) 이벤트 관리 로직 별도 구현 |
| 후순위채 | Subordinated Bond | 파산·청산 시 일반 채권보다 변제 순위가 낮은 채권. | 변제 순위(priority rank) 코드 관리. BIS 자기자본비율 계산 시 Tier 2 자본으로 인정 여부 플래그 필요 |
| ABS | Asset-Backed Securities | 자산(대출채권·매출채권 등)을 유동화하여 발행하는 증권. | 기초자산 풀(pool) 연결 구조 관리. 트랜치(tranche)별 우선순위·수익률 별도 테이블화 |
| MBS | Mortgage-Backed Securities | 주택담보대출채권을 기초자산으로 하는 ABS. 한국주택금융공사 발행이 대표적. | 조기상환율(CPR) 가정에 따라 현금흐름·듀레이션이 달라짐. CPR 시나리오 파라미터 관리 필요 |
| 물가연동국채 | Korea Inflation-Linked Bond / KTBi | 원금이 소비자물가지수(CPI)에 연동되는 국채. 실질금리 기반 이자 지급. | CPI 지수 시계열 DB 연동 필수. 원금 조정계수(index ratio) 계산: 결제일 CPI / 발행일 CPI, 소수점 6자리 관리 |
| 외화표시채권 | Foreign Currency Bond | USD·EUR 등 외화로 표시·상환되는 채권. 김치본드·아리랑본드 등 포함. | 통화 코드(ISO 4217) 필드 필수. 단가·이자 환산 시 기준환율(매매기준율) 적용 시점 명세화 필요 |
| 정크본드 (하이일드) | Junk Bond / High-Yield Bond | 신용등급 BB+(S&P 기준) 이하의 고위험·고수익 채권. | 등급 컷오프(BB+/Ba1) 기준으로 IG/HY 구분 플래그 자동 계산 로직 구현. 증거금률 별도 적용 |
| 커버드본드 | Covered Bond | 발행사 파산 시에도 담보자산으로 우선 변제받는 이중상환청구권부 채권. | 담보풀(cover pool) 연결 관리. 법적 근거(이중상환청구권법) 준수 여부 플래그 |

## 발행·구조

| 용어 | 영문/약어 | 뜻 | 개발 포인트 |
|------|-----------|-----|-------------|
| 액면가 | Face Value / Par Value | 채권에 기재된 명목 금액. 통상 10,000원(장내), 1,000원(장외) 단위. | 종목 마스터에 액면 단위(denomination) 필드 관리. 단가 계산 시 액면 기준 정규화 필요 |
| 발행가 | Issue Price | 채권을 최초 발행할 때의 매출 가격. 액면 이하(할인) 또는 이상(프리미엄) 가능. | 발행가와 액면가 차이를 상각(amortization) 처리. 유효이자율법 적용 시 발행가 기준 계산 |
| 표면금리 (쿠폰) | Coupon Rate | 액면가 대비 연간 이자 지급 비율. 채권 발행 시 확정되어 만기까지 유지(고정금리 기준). | 이자 계산: 액면가 × 쿠폰율 × 이자지급주기/365(또는 /2 반년). 관행상 원미만 절사 |
| 발행일 | Issue Date | 채권이 공식 발행된 날짜. 이자 산정의 기산점. | 이자 기산일은 발행일 기준. 단, 재개방(reopening) 시 기산일이 최초 발행일 유지됨에 주의 |
| 만기일 | Maturity Date | 원금이 상환되는 날짜. | 만기일이 영업일이 아닌 경우 차기 영업일 지급(Modified Following) 관행 적용. 달력(공휴일) DB 연동 필수 |
| 이자지급주기 | Coupon Frequency | 이자를 지급하는 주기. 연 1회(annual), 연 2회(semi-annual), 분기(quarterly), 월(monthly). | 주기 코드(A/S/Q/M)로 관리. 이자 일수 계산(day count) 주기와 연동 |
| 할인채 | Discount Bond / Zero-Coupon Bond | 이자를 별도 지급하지 않고, 액면가보다 낮은 가격으로 발행·만기 시 액면 지급. 통안채 단기물 등. | 이자 없음. 단가 = 액면 / (1 + r × D/365). 이자 계산 로직 분기 처리 필요 |
| 이표채 | Coupon Bond | 정기적으로 이자(쿠폰)를 지급하는 일반적인 채권 구조. | 현금흐름 스케줄(이자일·금액) 사전 생성 후 DB 저장. 이자 지급일 영업일 조정 처리 |
| 복리채 | Compound Interest Bond | 이자를 지급하지 않고 원금에 가산 복리 누적 후 만기 일시 상환. 국민주택채권 등. | 복리 계산: 원금 × (1 + r/m)^(n×m). 정밀도 소수점 10자리 이상 유지 후 최종 원미만 처리 |
| 단리채 | Simple Interest Bond | 이자를 단리로 계산하여 만기 일시 상환. | 단리 계산: 원금 × (1 + r × D/365). 경과일수 계산 시 발행일 포함/미포함 관행 확인 필수 |
| 변동금리부채권 | Floating Rate Note / FRN | 시장 기준금리(CD, SOFR, KOFR 등)에 스프레드를 더해 이자가 변동하는 채권. | 기준금리 참조(fixing date)와 이자 적용일(reset date) 구분 필수. 기준금리 시계열 DB 연동 |
| 콜옵션부 채권 | Callable Bond | 발행사가 만기 전 채권을 조기 상환할 수 있는 권리(콜옵션)가 부여된 채권. | 콜 일정(call schedule) 관리 테이블. 콜 행사 여부에 따라 현금흐름 시나리오 분기 처리 |
| 풋옵션부 채권 | Puttable Bond | 투자자가 만기 전 발행사에 채권 매도를 청구할 수 있는 권리가 부여된 채권. | 풋 행사 청구 이벤트 접수·처리 워크플로우 구현. 행사 기간(window) 유효성 검증 |
| 거치식 구조 | Grace Period Structure | 일정 기간 원금 상환 없이 이자만 지급하다가, 거치 기간 후 원금 상환이 시작되는 구조. | 거치 종료일 이후부터 원금 상환 스케줄 생성. 거치 기간 중 현금흐름: 이자만 포함 |
| 원금분할상환 | Amortizing Bond | 만기 일시 상환이 아닌 정기적으로 원금을 분할 상환하는 채권. | 상환 스케줄별 잔액 감소 반영. 단가·이자 계산 기준 잔액(outstanding principal) 사용 |
| 사모 발행 | Private Placement | 소수의 적격 투자자를 대상으로 비공개로 채권을 발행하는 방식. | 투자자 적격성 검증 로직. 공시 의무 면제 여부 플래그. 유통시장 제한 조건 관리 |
| 공모 발행 | Public Offering | 불특정 다수 투자자를 대상으로 공개 모집하는 채권 발행 방식. | 증권신고서 제출 이력 연결. 청약·배정·납입 일정 관리 |
| 재개방 (리오픈) | Reopening | 기존 발행 국채와 동일 조건(ISIN)으로 추가 발행하는 방식. 유동성 확대 목적. | 동일 ISIN에 발행 잔액 누적. 경과이자 기산일은 최초 발행일 유지 |
| Day Count Convention | 일수 계산 관행 | 이자 계산 시 분자(실제 일수)와 분모(연 기준일수)를 정하는 방식. Actual/365, Actual/360, 30/360 등. | 한국 국채: Act/365. 외화채: Act/360 또는 30/360 혼재. 종목별 day count 코드 필드 관리 필수 |

## 가격·수익률·평가

| 용어 | 영문/약어 | 뜻 | 개발 포인트 |
|------|-----------|-----|-------------|
| 만기수익률 | Yield to Maturity / YTM | 채권을 현재 가격에 매입하여 만기까지 보유할 때 얻는 내부수익률(IRR). 가장 대표적인 수익률 지표. | 뉴턴-랩슨 또는 이분법으로 반복 수렴 계산. 수렴 조건: 가격 오차 < 0.0001원. 최대 반복 100회 권장 |
| 현재수익률 | Current Yield | 연간 이자 / 현재 채권 가격 × 100. 단기 소득 지표. | 계산 단순. 단가 기준으로 실시간 제공. 클린프라이스 사용 여부 명세 필요 |
| 표면수익률 (쿠폰수익률) | Coupon Rate / Nominal Yield | 액면가 대비 연간 이자 비율. 발행 시 확정. | 종목 마스터 고정값. 변경 없음(FRN 제외) |
| 채권단가 | Bond Price | 채권 거래의 기준 가격. 한국 장내채권은 액면 10,000원 기준 소수점 2자리. | 장내: 원미만 2자리(예: 10,123.45원). 장외: 협회 공시 기준. 단가 정밀도 필드 타입 DECIMAL(12,4) 권장 |
| 클린프라이스 | Clean Price / Flat Price | 경과이자를 제외한 순수 채권 가격. 시장에서 호가·체결에 사용되는 기준 가격. | 거래 시 표시 가격. 더티프라이스 = 클린프라이스 + 경과이자 |
| 더티프라이스 | Dirty Price / Full Price | 클린프라이스에 경과이자를 포함한 실제 결제 가격. 투자자가 실제 지급하는 금액. | 결제 금액 계산 기준. 더티프라이스 = 클린프라이스 + 경과이자. 소수점 이하 원미만 절사 |
| 경과이자 | Accrued Interest / AI | 직전 이자 지급일 이후 결제일까지 경과한 기간에 해당하는 이자. | AI = 액면 × 쿠폰율 × 경과일/365. 경과일 계산: 직전 이자지급일 포함, 결제일 미포함(관행 확인) |
| 매매수익률 | Transaction Yield / Deal Yield | 실제 거래에서 적용된 수익률. 체결 시 기록되는 수익률. | 체결 데이터 저장 시 수익률·단가 동시 보관. 단가←→수익률 역산 검증 로직 구현 |
| 할인율 | Discount Rate | 미래 현금흐름을 현재가치로 환산하는 데 사용하는 이율. | PV = CF / (1+r)^t. 연속복리/이산복리 구분. 국채 평가는 반기 복리 관행이 다수 |
| 채권평가사 평가단가 | Evaluated Price (Bond Pricing Agency) | KIS채권평가·나이스피앤아이·한국자산평가 등 채권평가사가 매일 산출하는 기준단가. | 평가사별 단가 수신 후 종목별 저장. 시가평가 기준단가로 활용. 수신 지연 모니터링 알람 필수 |
| 시가평가 | Mark-to-Market / MTM | 보유 채권을 매일 시장 가격(평가단가)으로 평가하여 손익을 인식하는 방법. | 평가기준일 평가단가 × 보유수량 = 평가금액. 전일 대비 평가손익 계산 배치 처리 |
| 장부가평가 | Amortized Cost / Book Value | 취득원가에 할인·프리미엄을 유효이자율법으로 상각하여 평가하는 방법. | 유효이자율 계산 후 기간별 이자수익·장부가 조정 배치. 만기까지 누적 상각 테이블 사전 생성 권장 |
| 액면이자 | Face Interest / Par Interest | 액면가에 쿠폰율을 곱해 산출되는 이자 금액. | 이자 = 액면 × 쿠폰율 × 지급주기/12(개월 기준). 원미만 절사(절상 관행 확인) |
| YTC (콜수익률) | Yield to Call / YTC | 콜 행사일을 만기로 가정하고 계산한 YTM. 콜옵션부 채권의 대안 수익률 지표. | 콜 스케줄별 YTC 다중 계산. 가장 낮은 YTC = Yield to Worst(YTW)로 리스크 관리 기준 활용 |
| YTW (최악수익률) | Yield to Worst / YTW | 콜·풋·조기상환 등 모든 시나리오 중 투자자에게 가장 불리한(낮은) 수익률. | YTM과 모든 YTC 중 최솟값. 내재 옵션 행사 시나리오 전수 계산 필요 |
| 스팟레이트 (현물금리) | Spot Rate / Zero Rate | 현시점부터 특정 만기까지 단일 현금흐름에 적용되는 무위험 이자율. 수익률곡선 부트스트래핑으로 산출. | 국고채 현물금리 부트스트래핑 구현. 만기별 스팟레이트 테이블 일별 생성 |
| 포워드레이트 | Forward Rate | 미래 특정 시점부터 또 다른 시점까지의 선도 이자율. 스팟레이트로 역산. | f(t1,t2) = [(1+s2)^t2 / (1+s1)^t1]^(1/(t2-t1)) - 1. FRN 이자 예측 등에 활용 |
| OAS (옵션조정스프레드) | Option-Adjusted Spread / OAS | 내재 옵션을 제거한 순수 신용·유동성 스프레드. 옵션부 채권 비교에 사용. | 몬테카를로 또는 이항트리 모델 기반 계산. 계산 집약적 — 비동기 배치 처리 권장 |

## 위험·민감도

| 용어 | 영문/약어 | 뜻 | 개발 포인트 |
|------|-----------|-----|-------------|
| 듀레이션 | Duration | 채권의 평균 현금흐름 회수 기간이자 금리 변화에 대한 가격 민감도 지표. | 맥컬레이 듀레이션 = Σ(t × PV(CFt)) / 채권가격. 정밀도 소수점 4자리 이상 유지 |
| 맥컬레이 듀레이션 | Macaulay Duration | 현금흐름의 현재가치로 가중평균한 만기(단위: 년). 듀레이션의 원형 개념. | D_mac = [Σ t_i × CF_i / (1+y)^t_i] / P. 이표채 각 현금흐름별 루프 계산 |
| 수정듀레이션 | Modified Duration / MD | 금리 1% 변화 시 채권 가격 변화율(%). 맥컬레이 듀레이션 / (1 + YTM/m). | MD = D_mac / (1 + y/m). 가격 변화 근사: ΔP/P ≈ -MD × Δy. 리스크 시스템 핵심 지표 |
| 컨벡시티 | Convexity | 금리 변화에 따른 채권 가격의 곡률(비선형성). 수정듀레이션의 근사 오차를 보정. | 가격 변화: ΔP/P ≈ -MD×Δy + ½×Convexity×(Δy)². 소수점 2자리. 대규모 금리 변동 시 필수 |
| DV01 (PVBP) | Dollar Value of 01bp / Price Value of Basis Point | 금리 1bp(0.01%) 변화 시 채권 가격의 절대 변화액. 헤지·리스크 관리 핵심 지표. | DV01 = 채권가격 × MD × 0.0001. 단위: 원. 금리 헤지 비율 산출에 직접 활용 |
| 금리위험 | Interest Rate Risk | 시장금리 변동으로 채권 가격이 변하는 위험. | 듀레이션·컨벡시티·DV01로 수치화. VaR 산출 시 금리 변동성 시나리오 파라미터 관리 |
| 신용스프레드 | Credit Spread | 동일 만기 무위험 채권(국채) 대비 회사채 수익률의 초과분. 신용위험 프리미엄. | 스프레드 = 회사채 YTM - 국채 YTM(동일만기 보간). 보간법: 선형 또는 스플라인 |
| 신용등급 | Credit Rating | 신용평가사(한국신용평가·NICE신용평가·한국기업평가)가 부여하는 채무 상환 능력 등급. AAA~D. | 등급 코드 표준화(AAA/AA+/AA/AA-/.../D). 등급 변경 이력 테이블 관리. 등급 컷오프 자동 계산 |
| 신용위험 | Credit Risk | 채권 발행사가 이자 또는 원금을 지급하지 못할 위험. | 신용등급 기반 PD(부도확률) 매핑 테이블. 등급별 스프레드 커브 관리 |
| 유동성위험 | Liquidity Risk | 채권을 원하는 시점에 적정 가격으로 매도하지 못할 위험. | 거래량·호가 스프레드 기반 유동성 점수화. 비상장·사모채는 유동성 위험 가중치 상향 |
| 디폴트 (부도) | Default | 채권 발행사가 이자 또는 원금 지급 의무를 이행하지 못하는 사건. | 부도 이벤트 수신 시 해당 종목 전 계좌 동결 처리 자동화. 부도 일자·유형(기술적/실질) 기록 |
| 조기상환위험 | Prepayment Risk | MBS·ABS에서 기초자산 차주가 예상보다 일찍 상환하여 투자자 수익률이 낮아지는 위험. | CPR(연간조기상환율) 가정 파라미터 관리. PSA 모델 또는 자체 모델 선택 가능하도록 설계 |
| 재투자위험 | Reinvestment Risk | 받은 이자를 동일 수익률로 재투자하지 못할 위험. YTM 가정에 내재. | YTM 계산 시 재투자 가정 명시. 실현수익률(Realized Yield) 별도 계산 기능 제공 권장 |
| 금리민감도 | Interest Rate Sensitivity | 금리 변화에 대한 채권 포트폴리오의 총 노출. DV01·듀레이션으로 표현. | 포트폴리오 DV01 = Σ(종목별 DV01 × 보유수량). 일별 리스크 리포트 핵심 항목 |
| VaR | Value at Risk | 일정 신뢰구간(95%·99%)과 보유기간(1일·10일) 하에서 발생 가능한 최대 손실액. | 히스토리컬/파라메트릭/몬테카를로 방법 선택. 금리 VaR는 수익률 변동 시계열 필수 |
| CVA | Credit Valuation Adjustment | 거래상대방 신용위험을 반영한 파생상품·채권 가치 조정액. | CVA = Σ PD(t) × LGD × EE(t) × DF(t). 법인별 PD 곡선 관리 필요 |
| 신용평가사 | Credit Rating Agency / CRA | 한국신용평가(KIS), NICE신용평가, 한국기업평가(KR)의 국내 3사. S&P, Moody's, Fitch는 글로벌. | 평가사별 등급 체계 매핑 테이블(KIS↔S&P↔Moody's). 복수 등급 수신 시 최저 등급 사용 여부 정책 반영 |

## 거래·기타

| 용어 | 영문/약어 | 뜻 | 개발 포인트 |
|------|-----------|-----|-------------|
| 장내채권 | Listed Bond / Exchange-Traded Bond | 한국거래소(KRX)에 상장되어 거래소를 통해 거래되는 채권. | KRX FIX 메시지 연동. 체결 데이터 실시간 수신. 상장 종목 마스터 KRX OPEN API 연동 |
| 장외채권 | OTC Bond / Over-the-Counter Bond | 거래소 밖에서 금융기관 간 또는 기관-고객 간 직접 거래되는 채권. 대부분의 채권 거래. | KOFIA 채권 장외거래 보고 시스템 연동. 체결 후 15분 내 보고 의무. 보고 실패 모니터링 알람 |
| RP (환매조건부채권) | Repurchase Agreement / Repo | 채권 매도자가 일정 기간 후 정해진 가격으로 재매입(환매)할 것을 약정하고 채권을 매도하는 단기 자금 거래. | 매도·환매 2개 거래 생성. 이자 = 매도금액 × RP금리 × 기간/365. 담보채권 관리 테이블 필수 |
| 역RP | Reverse Repo | RP 거래의 반대편. 채권을 매입하고 일정 기간 후 재매도. 자금 운용 목적. | RP 매도 → 역RP 매입 관계 연결 키 관리. 마진콜(담보 부족) 트리거 로직 구현 |
| RP 금리 | Repo Rate | RP 거래에서 자금 차입비용을 나타내는 이자율. | 연율(annualized) 기준. 이자 계산: 거래금액 × 금리 × 일수/365. 원미만 절사 |
| 단순매매 | Outright Transaction | 채권을 단순히 매수 또는 매도하는 거래. 조건 없음. | 기본 거래 유형. 매수·매도 방향코드(B/S), 수량, 단가, 수익률 4요소 필수 저장 |
| 조건부매매 | Conditional Sale | RP처럼 매매에 환매 또는 재매도 조건이 부가된 거래. | 거래 유형 코드로 단순/조건부 구분. 조건 이행일 스케줄 관리 |
| 채권대차 | Bond Lending / Securities Lending | 채권 보유자가 일정 기간 채권을 빌려주고 대차료를 받는 거래. 공매도 등에 활용. | 대차 기간 중 쿠폰 이자 수령 주체(제조/운용) 명확화. 대차 기간 만료 자동 회수 처리 |
| 채권결제 | Bond Settlement | 채권 거래 후 채권과 대금을 교환하는 결제 과정. | T+1(국채 장내), T+2(회사채 장외) 등 상품별 결제주기(settlement cycle) 관리. 결제 불이행 모니터링 |
| DVP | Delivery vs. Payment | 채권 인도와 대금 지급을 동시에 처리하는 결제 방식. 결제 리스크 최소화. | 한국예탁결제원(KSD) 시스템과 연동. DVP 실패 시 미결제(fail) 처리 및 패널티 계산 |
| 한국예탁결제원 | Korea Securities Depository / KSD | 채권·주식 등 증권의 예탁·결제를 담당하는 중앙예탁기관. | KSD 전문 인터페이스(SWIFTNet/전용선) 연동. 종목 등록·말소 이벤트 실시간 수신 |
| 호가수익률 | Quoted Yield | 채권 매매 시 제시하는 수익률. 장외채권 거래의 기본 단위. | 수익률 ↔ 단가 역산 함수 구현. 틱 단위(0.001% 또는 0.01%) 유효성 검증 |
| 국고채전문딜러 | Primary Dealer / PD | 기획재정부가 지정한 국고채 입찰 참가 자격을 가진 금융기관. | PD 기관 코드 관리. 입찰 참가·낙찰 데이터 연동. PD 의무 이행 실적 관리 |
| 지표금리 | Benchmark Rate | 시장의 기준이 되는 금리. 한국에서는 국고채 3년·10년물이 대표적. | 지표 종목 플래그 관리. 지표 교체 시 이력 보존 필수 |
| 기준금리 | Base Rate / Policy Rate | 한국은행 금융통화위원회가 결정하는 단기 정책금리. 모든 금리의 기초. | 한은 금통위 결정일 캘린더 관리. 기준금리 변경 이벤트 수신 후 FRN 금리 재산출 트리거 |
| 수익률곡선 (일드커브) | Yield Curve | 동일 신용도 채권의 만기별 수익률을 연결한 곡선. 정상(우상향)/역전/플랫 형태. | 만기별 스팟·파 수익률 일별 생성·저장. 스플라인 보간(cubic spline) 또는 넬슨-시겔 모델 선택 |
| 부트스트래핑 | Bootstrapping | 이표채 수익률 데이터로부터 순차적으로 스팟레이트를 추출하는 방법. | 단기 → 장기 순서로 순차 계산. 이자지급 주기 불일치 시 보간 선행 필요 |
| 넬슨-시겔 모델 | Nelson-Siegel Model | 수준(level)·기울기(slope)·곡률(curvature) 3인자로 수익률곡선을 모델링하는 방법. | β0, β1, β2, λ 4개 파라미터 일별 추정. 최소제곱법(OLS/NLS) 적합. 추정 실패 시 전일 값 사용 |
| CD금리 | Certificate of Deposit Rate | 91일 만기 양도성예금증서 금리. FRN 기준금리로 널리 사용. | 금융투자협회 일별 고시값 수신. FRN 이자 재산출 트리거 연동 |
| KOFR | Korea Overnight Financing Rate | 한국은행이 산출하는 익일물 무위험지표금리(RFR). CD금리 대체 목적. | 일별 KOFR 수신·저장. KOFR 연동 FRN 이자 계산: 복리 누적(compound in arrears) 방식 |
| 자산운용사 운용지시 | Investment Instruction | 자산운용사가 채권 매매를 지시하는 문서/전문. 펀드 운용의 시작점. | 운용지시 접수→유효성 검증→주문 변환 워크플로우. 지시 번호 멱등성(idempotency) 보장 |
| 결제 실패 (Fail) | Settlement Fail | 결제일에 채권 또는 대금이 인도되지 않은 상태. 패널티(fail charge) 발생. | 결제일 자정까지 미결제 건 자동 감지. 패널티 계산: 미결제 금액 × 패널티율 × 실패일수 |
| 증거금 | Margin | 채권 거래 시 위탁증거금 또는 RP·파생 거래의 담보금. | 거래 유형별 증거금률 파라미터 테이블. 일중 MTM 변동에 따른 마진콜 자동 산출 |
